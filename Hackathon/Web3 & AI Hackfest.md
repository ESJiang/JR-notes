### neovim
> neovim可以理解成vim的更新版本. 支持异步插件, 可以不堵塞编辑流程, 为处理大项目带来更好的性能.

```shell
# macOS
brew install neovim
# Debian
apt install neovim
# edit ~/.zshrc
alias vim='nvim'
alias vi='nvim'
```
